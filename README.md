## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd https://github.com/SomeViet/portfolio-task-list
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
